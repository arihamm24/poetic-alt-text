# Poetic and Artistic Alt Text

## Key Questions

- What is Poetic Alt Text?
- How can we enable artists and curators to write poetic alt text?
- How can we use the interpretive process of alt text authorship to expand access to visual art?

## About this Resource

This is a digital adaptation and extension of the Alt Text as Poetry Workbook written by Shannon Finnegan and Bojana Coklyat as part of the “Alt Text as Poetry” Project.

## Community Partners

This website was developed for the Fulton County Public Arts Future Lab to complement their ADA 36th Anniversary Celebration.
